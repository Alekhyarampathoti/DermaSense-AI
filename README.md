# DermaSense AI 🩺

AI-powered dermatology assistant built using Gemini 2.5 Flash and Gradio.

## Features
- AI Skin Analysis
- Acne Severity Detection
- Personalized Skincare Guidance
- Professional Medical UI

## Tech Stack
- Python
- Gradio
- Gemini API
- PIL

## Run Locally

```bash
pip install -r requirements.txt
python app.py
